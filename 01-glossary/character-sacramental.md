# Character, Sacramental

## Catechism of the Catholic Church

An indelible spiritual mark which is the permanent effect of the Sacraments of Baptism, Confirmation, and [[holy-orders|Holy Orders]], by which a person is given a new permanent configuration to Christ and a specific standing in the Church; the reception of these sacraments is never repeated ([[2.2.1.0.p3n#^ccc-1272|1272]], [[2.2.1.2.p3s#^ccc-1304|1304]], [[2.2.3.6.p4y#^ccc-1582|1582]]).
